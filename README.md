## Installation

### Vim-Plug

```vim
Plug 'YSW0630/vscpp'
```

### Manual

Put `vscpp.vim` file in your `~/.vim/colors/` directory and add the following line to your `~/.vimrc`:

```vim
colorscheme vscpp
set laststatus=2 "Optional
```
